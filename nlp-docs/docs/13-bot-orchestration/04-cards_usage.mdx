# Cards Usage

As commented in [card](/bot-orchestration/dlg_commands_reference#card) section, npl.js keeps high compatibility with adaptive cards to allow displaying more custom content. 

## Example

Having this card file:

<details>
  <summary>boolean-card.json</summary>
    <pre>
      {JSON.stringify({
        "name": "booleanCard",
        "type": "message",
        "attachments": [
          {
            "contentType": "application/vnd.microsoft.card.adaptive",
            "content": {
            "$schema": "http://adaptivecards.io/schemas/adaptive-card.json",
            "type": "AdaptiveCard",
            "id": "booleanAnswer",
            "version": "1.0",
            "speak": "{{ questionText }} Please, select an option: yes / no",
            "body": [
              {
                  "type": "TextBlock",
                  "text": "Question",
                  "size": "Large",
                  "weight": "Bold"
              },
              {
                  "type": "TextBlock",
                  "text": " Please, select an option: ",
                  "isSubtle": true
              },
              {
                "type": "TextBlock",
                "text": "{{ questionText }}"
              }
            ],
            "actions": [
              {
                  "type": "Action.Submit",
                  "title": "Yes",
                  "data": 
                  {
                    "booleanAnswer": "yes"
                  }
              },
              {
                  "type": "Action.Submit",
                  "title": "No",
                  "data": 
                  {
                    "booleanAnswer": "no"
                  }
              }
            ]
          }
          }
        ]
      }, null, 2)}
    </pre>
</details>

And a dialog like this:
```
import boolean-card.json

dialog main
  say Hi
  set questionText "This is an example of yes / no question"
  card booleanCard
  ask booleanCard
  say you said {{ booleanAnswer }}
```

Will generate this conversation flow:
```
bot> Hi
bot> This is an example of yes / no question
bot> (displaying the card)
user> (selecting an option)
bot> you said <yes/no>
```


You can find some ideas and examples of cards in the [adaptive cards designer](https://adaptivecards.io/designer/).
